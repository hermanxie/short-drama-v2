# 动作提示词审核清单 v1

## 基础信息

```text
剧集：
分镜：
审核日期：
审核人：
```

## 必查项

- [ ] 已理解完整剧本或本集前后剧情，不是孤立写单镜。
- [ ] 已建立人物状态、伤势、道具、场景轴线和群演调度连续性。
- [ ] 保留 `星辰剧本到提示词.md` 的完整镜头结构，没有简化为摘要。
- [ ] 每个镜头都有上一镜承接点、本镜起幅、本镜落幅、下一镜过门钩子。
- [ ] 第一秒直接进入动作。
- [ ] 人物、服装、武器与参考图一致。
- [ ] 场景结构与参考图一致。
- [ ] 环境参与动作，不只是背景。
- [ ] 最终提示词中包含具体环境动态描述。
- [ ] 动作链包含起手、接触、受力反馈、换位或反击、封口结果。
- [ ] 命中点有短停、受力、位移或破坏反馈。
- [ ] 镜头能看清人物位置、方向和空间关系。
- [ ] 快节奏但不乱抖。
- [ ] 运镜有视觉冲击力，且服务动作可读性。
- [ ] 没有站桩摆拍。
- [ ] 没有空挥、特效盖点、纯氛围描述。
- [ ] 时间轴段落按剧情和动作密度自由切分，没有机械套固定段落。
- [ ] 有台词时已经写入对应时间轴段落。
- [ ] 台词包含人物、情绪、口型和面部肌肉动作。
- [ ] 台词只作为声音和口型，不会生成字幕或画面文字。
- [ ] 画面保持真人写实电影感和清晰高画质。
- [ ] 人物脸部和手部有柔和补光。
- [ ] 皮肤有真实质感和立体感，不塑料、不灰、不糊。
- [ ] 脸、眼、嘴、手、关键道具优先对焦。
- [ ] 眼神光、鼻梁、颧骨、嘴唇、手背和武器边缘高光可读。
- [ ] 背景提供纵深和环境互动，但不抢人物焦点。
- [ ] 伤势、脏污、道具状态连续。
- [ ] 每条单镜提示词都包含完整【全局风格与美学】。
- [ ] 每条单镜提示词都包含完整【高风险规避 / 反向约束】。
- [ ] 明确禁止字幕、台词字幕、旁白文字、说明文字、分镜编号、镜头参数文字、角色名标签、UI、水印、平台 logo、时间码、底部图例、箭头说明、构图辅助线、草图宫格边框。
- [ ] 剧本文字、台词、镜头参数、钉子4行和审核说明不会被渲染进最终画面。
