# 动作优先提示词工作流 v1

## 工作流目标

把一集脚本和参考图片转化为动作优先的高密度视频提示词。

本工作流只服务 `101_动作` 实验区，不修改上一级主项目 pipeline。

本工作流以 `03_templates/星辰剧本到提示词.md` 的完整结构为基础：先完成故事定位、核心视觉统一、角色统一设定和分镜规划清单，再逐镜输出“挂载、相机位置、相机朝向、角色朝向、构图锚点、起幅、落幅、连续性约束、提示词、镜头参数、抽针/甩拍/短爆、微表情、钉子4行”。新增规则只嵌入该结构，不另起简化版。

## 流程

```text
脚本 + 参考图
-> 输入登记
-> 全剧/本集通读
-> 故事定位
-> 核心视觉统一
-> 角色/场景/道具统一设定
-> 风格与画质继承
-> 动作强化
-> 分镜数量规划
-> 分镜规划清单
-> 镜头衔接表
-> 单镜动作蓝图
-> 即梦视频提示词
-> 钉子4行质检
-> 人工审核记录
```

## 生成顺序

1. 先读完整脚本，不直接写最终提示词。
2. 建立全剧/本集状态表：人物关系、阵营、情绪、伤势、道具、场景轴线、群演调度。
3. 提取每场冲突目标和人物动机。
4. 读取并确认图片参考，建立角色和场景统一设定。
5. 继承原有风格与画质优点，固定脸部和手部补光、皮肤质感、眼神光、轮廓光和关键道具高光。
6. 标记所有动作段和非动作段。
7. 对动作段逐个设计环境参与点和环境动态。
8. 规划镜头数量和每镜钉子画面。
9. 建立镜头衔接表：上一镜承接点、本镜起幅、本镜落幅、下一镜过门钩子。
10. 按剧情复杂度自由切分时间轴段落，执行“段落数量不限”，不得机械套固定 0-3/3-6/6-9/9-12/12-15。
11. 每镜附镜头参数、抽针/甩拍/短爆、微表情、钉子4行。
12. 有台词时必须写入对应时间轴，并标明人物、情绪、口型和面部肌肉动作；同时明确台词只作为声音和口型，不得生成字幕。
13. 每条最终视频提示词末尾加入完整的【全局风格与美学】、【环境动态】、【高风险规避 / 反向约束】和画面文字禁令；不得只写“遵守通用约束”。
14. 审核动作是否连贯、镜头是否衔接、环境是否参与、命中是否真实、人物光影质感是否达标、对白是否遗漏、画面文字禁令是否明确。

## 审核标准

动作镜头必须通过：

- 动作不空。
- 环境不空。
- 镜头不乱。
- 命中不虚。
- 受力不跳。
- 节奏不断。
- 封口有结果。
- 镜头之间能接住上一镜并推出下一镜。
- 时间轴段落按剧情自由切分。
- 每条最终提示词都有环境动态。
- 台词不遗漏，且不生成字幕。
- 画面无字幕、无文字、无 UI、无水印、无说明标记。
- 下一镜有钩子。
